# 90-Day DevOps Learning Plan

## Current Level

Working professional — currently employed as SDE2, aiming to extend development experience with application deploying expertise with practical DevOps operations.

## 90-Day Goals

1. Build and manage cloud infrastructure for a sample application using a major cloud platform.
2. Automate build and deployment pipelines to deploy applications end-to-end to the cloud.
3. Gain hands-on operational understanding of the full development-to-deployment lifecycle.

## Core DevOps Skills to Build

- Cloud infrastructure setup and management
- CI/CD pipeline creation and automation
- End-to-end application deployment and operational support

## Weekly Time Budget

- 2 hours daily on weekdays
- Total: 10 hours per week

## Focus Approach

- Prioritize consistent practice over trying to learn everything at once.
- Use small, repeatable projects to connect coding, infrastructure, automation, and deployment.
- Review progress weekly and adjust the plan based on what is working best.
